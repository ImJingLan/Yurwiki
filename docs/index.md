# Yurwiki - Yurik 的 知识库

欢迎来到 Yurik 的 知识库

## 什么是 Yurwiki ?

Yurwiki 是由 Yurik 和维基的英文单词 Wiki 复合而成，代表着 Yurik 的 Wiki（实际上是知识库）

## 这里会记录什么?

记录一些在捣鼓服务器、软件、开发过程中遇到的问题和解决方案

## 和[博客](//yurik.cafe)有什么区别

博客更多地会写一些长文章，也会更多地涉及生活方面，笔法会偏生活化

Yurwiki 则更多地记录技术内容，偏向硬核
